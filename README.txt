PKR BUDGET CALCULATOR — Install as an App (PWA)
================================================

WHAT'S IN THIS FOLDER (upload ALL of these files together):
  index.html             the app
  manifest.webmanifest   app name + icon settings
  sw.js                  makes it work offline
  icon-192.png           app icon (small)
  icon-512.png           app icon (large)


STEP 1 — PUT IT ONLINE (GitHub Pages, free)
-------------------------------------------
  1. Go to github.com  ->  New repository  ->  name it e.g.  budget
     ->  set it to Public  ->  Create repository.
  2. On the repo page:  Add file  ->  Upload files  ->  drag in ALL FIVE
     files from this folder  ->  Commit changes.
     (Keep the file names exactly as they are.)
  3. Settings  ->  Pages  ->  under "Branch" pick  main  and  / (root)  ->  Save.
  4. Wait about 1 minute. Your link appears at the top, like:
        https://YOUR-USERNAME.github.io/budget/


STEP 2 — INSTALL IT LIKE AN APP
-------------------------------
  On a LAPTOP / DESKTOP (Chrome or Edge):
    - Open your link. In the address bar, look for the Install icon
      (a small monitor with a down-arrow, or a ⊕). Click it -> Install.
    - The app opens in its own window and gets a Start-menu / Dock / desktop icon.
      Launch it any time like a normal program.

  On a PHONE:
    - Android (Chrome):  menu (⋮)  ->  "Install app" / "Add to Home screen".
    - iPhone (Safari):  Share button  ->  "Add to Home Screen".


GOOD TO KNOW
------------
  - After the first open, the app works OFFLINE.
  - Your data is saved on EACH device by itself — your phone and laptop do
    NOT auto-sync. Use the in-app Excel export to back up or move data.
  - To update later: just upload a new index.html to the same repo.
    Installed copies refresh themselves automatically.
